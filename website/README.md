# Games Website

A simple React + Vite landing page for browser games.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The built site is emitted to `dist/`, which can be deployed directly on Vercel.

## Adding another game

Edit `src/main.jsx` and add another object to the `games` array with:

- `title`
- `eyebrow`
- `description`
- `image`
- `url`
- `tags`
